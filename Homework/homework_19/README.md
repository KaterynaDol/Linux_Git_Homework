# Homework 19 — Работа с файлами и правами

## Описание
В этом задании я работала с файлами и правами доступа в Linux.  
Нужно было написать два скрипта:
1. Скрипт для создания 100 файлов со случайными именами.
2. Скрипт для переноса файлов с чётными именами в отдельную директорию.

---

## Ход выполнения

1. Создала папку `source` и `even`.
2. Написала скрипт `create_files.sh`, который с помощью цикла `{1..100}` и переменной `$RANDOM`
   создаёт 100 файлов с случайными числовыми именами.
3. Написала скрипт `move_even_files.sh`, который:
   - получает список файлов из `source`,
   - проверяет, что имя файла — число,
   - вычисляет остаток от деления имени на 2,
   - если число чётное — переносит файл в `even`.

---

## Использованные команды
`mkdir`, `cd`, `touch`, `for`, `if`, `mv`, `echo`, `chmod`.
